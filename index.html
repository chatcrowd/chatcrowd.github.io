chatcrowd(ViL): A Dialog-based Platform
for Visual Layout Composition



Chat-crowd is an interactive environment for visual layout composition via conversational interactions. We expect that this system will be useful to build multimodal goal-oriented dialog tasks that require spatial and geometric reasoning.
